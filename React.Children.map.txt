React.Children.map là một function bên trong React.Children, là một object chứa một số functions cho phép
dễ dàng làm việc với components properties this.props.children, nó hoàn thành một function trên
mỗi children trực tiếp của component và trả về một object.

React.Children.map( children, myFn [, context])
ở đây children là một object mà chứa các childrens mục tiêu, myFn là function được chấp nhận tới
mỗi children, context là optional và sẽ set this trên mapping function

ví dụ:

var MyComponent = React.createClass({
  render: function(){
    React.Children.map(this.props.children, funciton(child) {
      console.log(child);
    });
    return (
      <div>{this.props.name}</div>
    );
  }
});

React.render(<MyComponent name="frodo">
  <p key="firsty">a child</p>
  <p key="2">another</p>
</MyComponent>, document.getElementById('container'));
